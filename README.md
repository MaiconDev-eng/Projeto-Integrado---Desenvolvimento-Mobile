### 📱 CampoLog – Aplicativo Mobile para Gestão Agrícola

O **CampoLog** é um aplicativo mobile desenvolvido como projeto acadêmico no curso de Análise e Desenvolvimento de Sistemas, com o objetivo de otimizar o controle de aplicações agrícolas no campo.

A solução foi criada para atender às necessidades da empresa AgroZaneratto, que enfrentava dificuldades com registros manuais, como perda de informações, erros e falta de organização dos dados.

Vídeo : https://www.youtube.com/watch?v=e53HKqVB-Hs

### 🚜 Principais funcionalidades

* Cadastro de usuários
* Registro de aplicações por operador e talhão
* Consulta de histórico de atividades
* Geração de relatórios
* Funcionamento **offline**, com sincronização de dados quando há conexão com a internet

### 💡 Diferenciais

* Operação mesmo sem acesso à internet (ideal para ambiente rural)
* Interface simples e intuitiva
* Organização eficiente das informações
* Redução de erros e aumento da confiabilidade dos dados

### 🛠️ Tecnologias utilizadas

* Flutter
* Dart
* Integração com backend e banco de dados

### 🎯 Objetivo

Digitalizar e melhorar a gestão das atividades agrícolas, proporcionando maior controle, eficiência operacional e suporte na tomada de decisões.
